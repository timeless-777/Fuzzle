# Fuzzle

## Site

- [https://fuzzle-link.vercel.app](https://fuzzle-link.vercel.app)

## Start

```bash
cd src/web
```

- Install

```bash
bun install
```

- Copy .env.sample to .env

```bash
cp .env.sample .env
```

- Start

```bash
bun run dev
```

## Tech

- Next.js
- Supabase
- Neynar
- Tailwind CSS
- Vercel
